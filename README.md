# GLM

This is a mobile-first development repository managed entirely from an iPhone 15 using GitHub Actions and the GLM coding agent.

## How to use the AI Agent
To ask GLM to write or update code in this repository:
1. Open a new **Issue** or **Pull Request** in the GitHub mobile app.
2. Type your prompt and mention **@claude** at the beginning.
3. The GLM engine will run via GitHub Actions, write the code, and commit it here automatically.

## Project Contents
* `.github/workflows/glm.yml` — The automation script powering the AI agent.
